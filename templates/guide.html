Guide
Start by logging into your account. Use your admin credentials to access the system. Once logged in, you will be redirected to the main dashboard where you can choose from various Wi-Fi security audit features.
After logging in, click on the Scan Networks button. This will detect available Wi-Fi networks around you, providing a list of networks with their details.
Click on the Connected Network button to analyze the details of your current Wi-Fi connection. This will provide you with insights into your network’s security status.
Next, click on Check Vulnerabilities to identify any security risks in the detected networks. The system will alert you to issues like weak encryption, rogue access points, and unsecured networks.
Use the Audit Security feature to run a detailed security audit on your selected network. The tool will generate a comprehensive report about the network’s security level and any areas for improvement.
You can also perform a Speed Test to measure the performance of your Wi-Fi connection. Click on the Run Speedtest button to check download and upload speeds, latency, and more.
After completing the audit, you can generate a Wi-Fi Security Audit Report. This report will summarize:
  • Connected Network Details: Information about your current Wi-Fi connection.
  • Security Vulnerabilities: Any risks like weak encryption or unsecured networks.
  • Wi-Fi Speed Test Results: Data on download speed, upload speed, and ping.
  • Security Recommendations: Suggestions for improving Wi-Fi security.
The report will be available as a downloadable PDF.
